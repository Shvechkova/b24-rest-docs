> Кто может выполнять метод: любой пользователь

Метод `crm.activity.fields` возвращает описание полей активности.

## Параметры

Без параметров

## Примеры

{% list tabs %}

- JS

    ```js
    BX24.callMethod(
        "crm.activity.fields",
        {},
        function(result)
        {
            if(result.error())
                console.error(result.error());
            else
                console.dir(result.data());
        }
    );
    ```

{% endlist %}
